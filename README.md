# byshakir

Metropolitan Luxury Interior Design Portfolio - A premium showcase website for high-end interior design projects.

## Tech Stack

- Astro + Tailwind CSS
- TypeScript
- View Transitions

## Development

```bash
# Install dependencies
npm install

# Run dev server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Deployment

Deployed via Cloudflare Pages. Pushes to `main` automatically deploy to production.
